# TCBP - Total Commander Batch Python
This tool is a Python re-implementation of similar functionality, inspired by TCBL (Total Commander Batch Builder & Launcher).
이 도구는 TCBL(Total Commander Batch Builder & Launcher)에 영감을 받아, 유사한 기능을 파이썬으로 재구축한 것입니다.
토탈커맨더 등 파일관리자와 연동하여 배치명령 처리를 도와줍니다.
